# 🌐 AskDevPK — Personal Developer Platform

**AskDevPK** is my personal developer website and portfolio platform where I showcase projects, technical work, experiments, and development progress.

The platform acts as a **central hub for my development journey**, including projects, blogs, services, and system architecture experiments.

🌍 Live Website
https://www.askdevpk.me

---

# 🚀 About The Project

AskDevPK was built to create a **professional digital presence** where I can organize and present my work as a developer.

The platform includes sections for:

* Developer portfolio
* Project showcase
* Technical blog posts
* Development services
* Contact and collaboration

It also serves as a **continuous learning platform**, where new projects, improvements, and experiments are regularly added.

# ⚙️ Installation

Clone the repository

```
git clone https://github.com/itspksharma/askdevpk.git
```

Go to the project directory

```
cd askdevpk
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```
